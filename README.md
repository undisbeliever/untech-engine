UnTech Game Engine
==================

A overly-complicated homebrew platformer game engine designed for a Super Nintendo.


**ENGINE STATUS**: Stalled.  The MetaSprite subsystem a major bottleneck and needs to be redesigned and replaced.


NOTICE
======

The untech-engine is still in the design/experimentation phase.  Expect
major and unexpected changes to the engine API.

Build Requirements
===================
 * [my private fork of bass](https://github.com/undisbeliever/bass-untech)
   (included as a git submodule)
 * [untech-editor](https://github.com/undisbeliever/untech-editor)
   (included as a git submodule)
 * python3
 * gnu Make
